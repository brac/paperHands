# Check-in dashboard (read-only)

A small Vite + React SPA that renders a recorded backtest run for the ETF rebalancer. It
**reports — it never trades.** Benchmark is SPY; the goal is to match it with less risk, not
to beat it.

## Use

1. Record a run (the backtest harness writes to `results.sqlite`):
   ```
   python -m runner.backtest --start 2018-01-01 --end 2023-12-31 --mode rebalance
   ```
2. Export it to static JSON the SPA reads:
   ```
   python -m dashboard.export --latest        # writes dashboard/web/public/data.json
   ```
3. Run the SPA:
   ```
   cd dashboard/web
   npm install
   npm run dev      # or: npm run build && npm run preview
   ```

`public/data.json` is regenerated by the export command; the committed copy is just a
placeholder so the app loads before the first export.

## What it shows

- **Equity curve vs buy-and-hold SPY** over the run window.
- **Positions vs target weights** with per-asset drift.
- **Rebalance trades + realized gain/loss** (average-cost; makes taxable events visible).
- **Risk stats** — total/CAGR return, volatility, max drawdown, Sharpe, turnover — portfolio
  beside SPY, so "match with less risk" is checkable at a glance.
